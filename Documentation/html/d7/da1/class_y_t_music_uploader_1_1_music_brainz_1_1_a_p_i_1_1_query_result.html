Abstract base class for MusicBrainz queries returning lists (with paging support).

Property Documentation

◆ Count

int YTMusicUploader.MusicBrainz.API.QueryResult.Count
getset

Gets or sets the total list items count.

This might be different form the actual list items count. If the list was generated from a search request, this property will return the total number of available items (on the server), while the number of returned items is limited by the requests 'limit' parameter (default = 25).
