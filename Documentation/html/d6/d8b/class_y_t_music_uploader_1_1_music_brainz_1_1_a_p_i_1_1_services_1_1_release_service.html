Browse all the releases in the MusicBrainz database, which are directly linked to the entity with given id.

Parameters
entityThe name of the related entity.
idThe id of the related entity.
typeIf releases or release-groups are included in the result, filter by type (for example 'album').
statusIf releases are included in the result, filter by status (for example 'official', default = null).
limitThe maximum number of releases to return (default = 25).
offsetThe offset to the releases list (enables paging, default = 0).
incA list of entities to include (subqueries).
Returns

See http://musicbrainz.org/doc/Development/XML_Web_Service/Version_2#Release_Type_and_Status for supported values of type and status.
